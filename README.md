# 8 Levels of Agentic Engineering

An interactive assessment tool for the framework originally written by [Bassim Eledath](https://www.bassimeledath.com/blog/levels-of-agentic-engineering).

**[→ Take the assessment](https://fuentes-studio.github.io/agentic-levels)**

## What it is

A single-file HTML tool that helps engineers assess where they are in the 8-level agentic engineering progression — from tab complete to autonomous agent teams.

Each level includes:
- What it looks like when you're there
- What's holding you back
- A concrete next step
- Resources to close the gap

## Sharing

After taking the assessment, hit **Share Result** to copy a link. Recipients land on a recap card showing your level with a direct link to take it themselves.
